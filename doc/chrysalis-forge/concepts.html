4.1 SignaturesðŸ”—â„¹

Signatures define the typed interface for LLM tasks, similar to function signatures in typed languages.

struct

(struct Signature (name ins outs)
    #:extra-constructor-name make-Signature)
  name : symbol?
  ins : list?
  outs : list?
A typed interface for an LLM task.

ins — Input fields the model receives. outs — Output fields the model should produce.
