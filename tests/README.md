# Tests

This directory contains the test suite for Chrysalis Forge.

## Files

- **`run-tests.rkt`**: The main test runner. Execute with `racket tests/run-tests.rkt`.
- **`test-acp-tools.rkt`**: Validation for the tool system.
- **`test-multimodal.rkt`**: Tests for vision/image capabilities.
- **`test-pricing-model.rkt`**: Tests for cost calculation and pricing retrieval.
- **`test-session.rkt`**: Validates session management and context switching.
- **`test-utils-retry.rkt`**: Tests the auto-correction loop.
- **`test-utils-time.rkt`**: Time handling tests.
