# Source Root

The core source code of Chrysalis Forge is organized into functional subdirectories.

## Directory Structure

- **`core/`**: Central agent logic and optimization engines (GEPA, Meta-GEPA).
- **`llm/`**: LLM interaction and DSPy-style programming model.
- **`stores/`**: Persistence and state management (SQLite, RDF, Vector).
- **`tools/`**: Tool implementations (Filesystem, Git, Web, MCP).
- **`utils/`**: General helper modules.

For detailed information, see the README in each subdirectory.
