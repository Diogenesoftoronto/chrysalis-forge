# Utilities

Helper modules and cross-cutting concerns for Chrysalis Forge.

## Modules

- **`debug.rkt`**: Configurable logging system with categories and levels.
- **`dotenv.rkt`**: Environment variable management.
- **`test-gen.rkt`**: Logic for LLM-powered test generation.
- **`utils-retry.rkt`**: Implementation of the auto-correction loop for code execution.
- **`utils-spinner.rkt`**: CLI UI feedback (spinners).
- **`utils-time.rkt`**: Time-related utility functions.
